# 🚀 Playwright Automation Framework

## 📌 Project Overview
This project is an end-to-end test automation framework built using Playwright with TypeScript.

---

## 🛠️ Tech Stack
- Playwright
- TypeScript
- Node.js

---

## 📂 Features
- Page Object Model (POM)
- API Testing
- PDF Validation
- Auto-waiting & Assertions

---

## ▶️ How to Run
```bash
npm install
npx playwright test
